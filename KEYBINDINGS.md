# 快捷键说明（新手版）

如果你刚开始用这套桌面，先记住“必会 13 个”，其余按需查表。

`Mod` 在本仓库的 Niri 会话中等同 `Super`（Windows 键）；在 winit 窗口环境中等同 `Alt`。

---

## 1. 先记这 13 个

| 快捷键 | 功能 |
|---|---|
| `Mod + Return` | 打开终端 |
| `Mod + Space` | 打开应用启动器（Fuzzel） |
| `Mod + E` | 打开文件管理器 |
| `Mod + Q` | 关闭当前窗口 |
| `Mod + Left/Right` | 切换左右窗口焦点 |
| `Mod + 1..9` | 切到工作区 1..9 |
| `Mod + C` | 浮动/平铺切换 |
| `Mod + V` | 在浮动窗口与平铺窗口间切换焦点 |
| `Mod + X` | 全屏当前窗口 |
| `Print` | 截图 |
| `XF86AudioRaiseVolume/LowerVolume` | 音量增减 |
| `Mod + Shift + L` | 锁屏 |
| `Mod + Shift + E` | 退出 Niri（有确认） |

## 2. 常用桌面快捷键

### 应用与会话

| 快捷键 | 功能 |
|---|---|
| `Mod + Return` | 打开终端 |
| `Mod + Shift + Return` | 浮动终端 |
| `Mod + Space` | 打开应用启动器（Fuzzel） |
| `Mod + Shift + Slash` | 显示快捷键提示层 |
| `XF86Search` | 打开应用启动器（Fuzzel） |
| `Mod + E` | 打开文件管理器 |
| `Mod + Shift + L` | 锁屏 |
| `Mod + O` | 列 tabbed 显示切换 |
| `Mod + Ctrl + E` | 打开会话菜单（wlogout） |
| `Mod + Shift + E` | 退出 Niri（有确认） |
| `Ctrl + Alt + Delete` | 退出 Niri（有确认） |

### 窗口导航与移动

| 快捷键 | 功能 |
|---|---|
| `Mod + Left/Down/Up/Right` | 左右列切换；上下切换同列窗口/工作区 |
| `Mod + H/J/K/L` | 左右移动列；上下移动窗口 |
| `Mod + Ctrl + Left/Down/Up/Right` | 同上（方向键版本） |
| `Mod + Home/End` | 聚焦最左/最右列 |
| `Mod + Ctrl + Home/End` | 将当前列移动到最左/最右 |
| `Mod + ,/.` | 在列内/列外切换（并入或移出窗口） |
| `Mod + D/F` | 吸入右侧窗口到当前列 / 将当前列底部窗口移到右侧列 |
| `Mod + C` | 浮动/平铺切换 |
| `Mod + V` | 在浮动窗口与平铺窗口间切换焦点 |
| `Mod + S/G` | 将当前窗口下移 / 上移 |
| `Mod + W` | 打开/关闭窗口概览 |
| `Mod + X` | 全屏当前窗口 |
| `Mod + Q` | 关闭当前窗口 |

### 多显示器

| 快捷键 | 功能 |
|---|---|
| `Mod + Shift + Alt + H/J/K/L` | 聚焦左/下/上/右显示器 |
| `Mod + Ctrl + H/J/K/L` | 将窗口移动到左/下/上/右显示器 |
| `Mod + Ctrl + Shift + H/J/K/L` | 将工作区移动到左/下/上/右显示器 |

### 工作区

| 快捷键 | 功能 |
|---|---|
| `Mod + Page_Down/Page_Up` | 切换到下一个/上一个工作区 |
| `Mod + Shift + Page_Down/Page_Up` | 移动当前工作区到下一个/上一个位置 |
| `Mod + Alt + Left/Right` | 切换到上一个/下一个工作区 |
| `Mod + Alt + Down` | 跳到最底部空工作区（索引 99） |
| `Mod + Tab` | 在当前与上一个工作区间切换 |
| `Mod + 1..9` | 直达工作区 1..9 |
| `Mod + Ctrl + 1..9` | 将当前列移动到工作区 1..9 |
| `Mod + Alt + 1..9` | 将当前窗口移动到工作区 1..9 |
| `Mod + N/M` | 将当前窗口移动到上一个/下一个工作区 |
| `Mod + Ctrl + Page_Down/Page_Up` | 将当前列移动到下一个/上一个工作区 |
| `Mod + Shift + N` | 跳到最底部空工作区（新工作区入口） |
| `Mod + Ctrl + Shift + N` | 同上（备用组合） |
| `Mod + WheelScrollDown/Up` | 切换工作区（滚轮） |
| `Mod + Ctrl + WheelScrollDown/Up` | 移动列到下/上一个工作区（滚轮） |
| `Mod + WheelScrollRight/Left` | 聚焦右/左侧列（滚轮） |
| `Mod + Ctrl + WheelScrollRight/Left` | 移动列到右/左（滚轮） |

### 布局与尺寸

| 快捷键 | 功能 |
|---|---|
| `Mod + R` | 循环列宽预设 |
| `Mod + Shift + R` | 循环窗口高度预设 |
| `Mod + Ctrl + R` | 重置窗口高度 |
| `Mod + - / =` | 列宽 -10% / +10% |
| `Mod + Shift + - / =` | 同列窗口高度 -10% / +10% |
| `Mod + Z` | 当前列最大化 |
| `Mod + Shift + M` | 当前窗口最大化到边缘 |
| `Mod + Ctrl + Shift + F` | 窗口化全屏（fake fullscreen） |
| `Mod + Ctrl + F` | 列扩展到可用剩余宽度 |
| `Mod + Shift + C` | 当前列居中 |
| `Mod + Ctrl + Shift + C` | 可见列整体居中 |
| `Mod + Alt + Space` | 切换到下一个输入布局 |
| `Mod + Alt + Shift + Space` | 切换到上一个输入布局 |

### 截图与剪贴板

| 快捷键 | 功能 |
|---|---|
| `Print` | 截图（默认行为） |
| `Ctrl + Print` | 当前屏幕截图 |
| `Alt + Print` | 当前窗口截图 |
| `Mod + A` | 区域截图（保存并复制） |
| `Mod + Ctrl + C` | 剪贴板历史 |
| `Mod + Ctrl + B` | 发送信号给 waybar（刷新/切换可见性脚本） |
| `Mod + Ctrl + S` | 打开音量控制（pavucontrol） |

### 媒体与亮度

| 按键 | 功能 |
|---|---|
| `XF86AudioRaiseVolume/LowerVolume` | 音量增减（锁屏可用） |
| `XF86AudioMute` | 静音（锁屏可用） |
| `XF86AudioMicMute` | 麦克风静音（锁屏可用） |
| `XF86AudioPlay/Stop/Prev/Next` | 媒体控制（锁屏可用） |
| `XF86MonBrightnessUp/Down` | 屏幕亮度（锁屏可用） |
| `XF86KbdBrightnessUp/Down` | 键盘背光亮度（锁屏可用） |

### 会话控制与安全

| 快捷键 | 功能 |
|---|---|
| `Mod + Escape` | 切换 keyboard shortcuts inhibit（远程/KVM 逃生键） |
| `Mod + Shift + E` | 退出 Niri（有确认） |
| `Mod + Shift + P` | 关闭显示器电源（任意输入唤醒） |

## 3. Tmux（Prefix: `Ctrl + B`）

| 快捷键 | 功能 |
|---|---|
| `Ctrl + B` | Prefix |
| `Prefix + W` / `E` | 水平/垂直分屏 |
| `Prefix + Left/Down/Up/Right` | pane 焦点切换 |
| `Prefix + Q` | 关闭 pane |
| `Prefix + C` | 新建 window |
| `Prefix + D` / `S` | 下一个/上一个 window |
| `Prefix + G` | detach |
| `Prefix + R` | 重载 tmux 配置 |

---

## 4. Zellij（Tmux Mode）

| 快捷键 | 功能 |
|---|---|
| `Ctrl + B`（Normal） | 进入 Tmux mode |
| `Tmux mode + H/J/K/L` | pane 焦点切换 |
| `Tmux mode + S` / `V` | 分屏 |
| `Tmux mode + X` | 关闭 pane |
| `Tmux mode + C` | 新建 tab |
| `Tmux mode + W` / `Q` | 下一个/上一个 tab |
| `Tmux mode + G` | detach |
