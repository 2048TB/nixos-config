# Binary Cache 优化说明

本仓库在 `nix/modules/system.nix` 中显式配置以下二进制缓存：

- `https://nix-community.cachix.org`
- `https://nixpkgs-wayland.cachix.org`
- `https://cache.garnix.io`

另外，`https://cache.nixos.org` 为 Nix 默认官方缓存。

对应公钥通过 `nix.settings.trusted-public-keys` 固定。

## 建议实践

1. 优先使用 `just flake-check` 和 `just check` 进行改动验证，减少无效构建。
2. 升级依赖时先执行 `just update-nixpkgs`，验证通过后再全量 `just update`。
3. 出现意外本地编译时，先检查是否引入了未缓存包或 overlay 变更。

## 快速检查命令

```bash
# 查看是否会触发大量本地构建
nix build --dry-run path:/persistent/nixos-config#nixosConfigurations.zly.config.system.build.toplevel

# 进行 flake 级别检查
nix flake check path:/persistent/nixos-config
```
