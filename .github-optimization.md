# Binary Cache 优化说明

本仓库已在 `nix/modules/system.nix` 中配置以下二进制缓存：

- `https://cache.nixos.org`
- `https://nix-community.cachix.org`
- `https://nixpkgs-wayland.cachix.org`
- `https://cache.garnix.io`

对应公钥通过 `nix.settings.trusted-public-keys` 固定。

## 建议实践

1. 优先使用 `just flake-check` 和 `just check` 进行改动验证，减少无效构建。
2. 升级依赖时先执行 `just update-nixpkgs`，验证通过后再全量 `just update`。
3. 出现意外本地编译时，先检查是否引入了未缓存包或 overlay 变更。
