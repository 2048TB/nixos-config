# Binary Cache 说明（新手版）

这个仓库已经配置了二进制缓存，用来减少本地编译时间。

当前缓存：
- `https://nix-community.cachix.org`
- `https://nixpkgs-wayland.cachix.org`
- `https://cache.garnix.io`

---

## 1. 配置位置

- `flake.nix`：`nixConfig.extra-substituters` 与 `extra-trusted-public-keys`
- `hosts/outputs/default.nix`：运行期模块使用的缓存配置

---

## 2. 如何确认缓存正常

```bash
just flake-check
nix flake metadata path:/persistent/nixos-config
```

---

## 3. 常见问题

### 问题：还是大量源码编译

检查：
1. 网络是否可访问缓存域名
2. `flake.nix` 和 `hosts/outputs/default.nix` 的缓存配置是否一致
3. 缓存服务是否临时不可用（稍后重试）
